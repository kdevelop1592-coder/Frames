# 프레임 배우기 - 애니메이션의 원리

중학생을 위한 프레임(Frame)과 FPS(Frames Per Second) 개념 학습 웹사이트입니다. 스프라이트 애니메이션을 통해 프레임의 원리를 시각적으로 이해할 수 있습니다.

## 🎯 학습 목표

- 프레임(Frame)의 개념 이해
- FPS(Frames Per Second)의 의미 파악
- 프레임 속도가 애니메이션에 미치는 영향 체험
- 실생활 속 다양한 FPS 사례 학습

## 🚀 주요 기능

- **인터랙티브 애니메이션**: 실시간으로 FPS를 조절하며 애니메이션 변화 관찰
- **프레임 단위 제어**: 이전/다음 버튼으로 각 프레임 확인
- **성능 모니터링**: 실제 렌더링 FPS 표시
- **키보드 지원**: 스페이스바(재생/일시정지), 화살표(프레임 이동)
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업, Canvas API
- **CSS3**: CSS Variables, Grid, Flexbox, Glassmorphism
- **JavaScript**: ES6+, requestAnimationFrame
- **폰트**: Google Fonts (Noto Sans KR, Inter)

## 🎮 사용 방법

1. **FPS 조절**: 슬라이더를 움직여 1-60 FPS 사이에서 속도 변경
2. **재생/일시정지**: 중앙 버튼 클릭 또는 스페이스바
3. **프레임 이동**: 좌우 버튼 클릭 또는 화살표 키
4. **체험 학습**: 다양한 FPS에서 애니메이션 차이 비교
